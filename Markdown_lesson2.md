Чтобы добавить ненумерованые списки необходимо пункты выделить (*).
Например вот так:
* Элемент 1
* элемент 2
* Элемент 3
Чтобы пронумеровать нумерованные списки их необходимо просто пронумеровать
Например вот так:
1.первый пункт
2.второй пункт
3.третий
